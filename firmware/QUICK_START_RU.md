# 🌱 ШПАРГАЛКА: ESP32-C3 + Датчики влажности v1.2

## 📌 Быстрая распиновка

| Что | Куда | GPIO |
|-----|------|------|
| **Датчик 1** A0 | D4 | GPIO4 |
| **Датчик 2** A0 | D5 | GPIO5 |
| **Датчики** VCC | 3.3V | - |
| **Датчики** GND | GND | - |
| **Аккумулятор +** | VIN | - |
| **Аккумулятор -** | GND | - |

## 🔋 Аккумулятор 18650

```
18650 (+) ──────> VIN на ESP32-C3
18650 (-) ──────> GND на ESP32-C3
```

⚠️ **Используйте плату защиты TP4056!**

## 🔧 Настройка IntelliSense (VS Code)

Если видите ошибки `include not found`:

### Вариант 1: PlatformIO (рекомендуется)
1. Установите расширение **PlatformIO IDE**
2. Откройте папку `firmware` в VS Code
3. PlatformIO автоматически настроит IntelliSense

### Вариант 2: Arduino IDE
1. Установите расширение **Arduino**
2. Откройте `esp32c3_soil_sensor.ino` в Arduino IDE
3. Выберите плату: **ESP32C3 Dev Module**

### Вариант 3: Только IntelliSense
Файлы `.clangd`, `.ccls`, и `.vscode/settings.json` уже настроены!

## ⚙️ Настройки для редактирования

Откройте `esp32c3_soil_sensor.ino` и измените:

```cpp
const char* WIFI_SSID = "ВАШ_WIFI";
const char* WIFI_PASSWORD = "ВАШ_ПАРОЛЬ";

// Локальный сервер (укажите IP вашего сервера)
const char* SERVER_URL = "http://192.168.1.100:8000/api/telemetry";

// Продакшен
// const char* SERVER_URL = "http://aqllidala.uz/api/telemetry";

const String DEVICE_ID = "ESP32C3_FIELD_01";  // Должен совпадать с field_id в БД
```

> **Важно:** Backend ожидает `/api/telemetry` с полями:
> `field_id`, `moisture`, `temperature`, `battery`

## 🔧 Калибровка

1. Загрузите `calibration_test.ino`
2. Откройте Serial Monitor (115200)
3. Запишите значения:
   - В воздухе → `CALIBRATION_DRY`
   - В воде → `CALIBRATION_WET`
4. Вставьте в основную прошивку

## 💾 Загрузка

1. Подключите ESP32-C3 через USB-C
2. Выберите плату: **ESP32C3 Dev Module**
3. Нажмите **Upload**
4. Если не идет → зажмите **BOOT** + **RESET**

## 📊 Потребление

| Режим | Ток |
|-------|-----|

| Глубокий сон | ~10-50 μA |
| **От 18650** | **6-12 месяцев** |

## ✅ Чеклист перед запуском

- [ ] WiFi SSID и пароль правильные
- [ ] Датчики подключены к GPIO4 и GPIO5
- [ ] VCC датчиков → 3.3V (не 5V!)
- [ ] GND всех компонентов соединены
- [ ] Аккумулятор через плату защиты
- [ ] Полярность аккумулятора проверена (+ к VIN, - к GND)

## 🆘 Проблемы

| Проблема | Решение |
|----------|---------|
| Не грузится | Зажать BOOT при подключении USB |
| Нет показаний | Проверить GPIO4/GPIO5 и GND |
| 0% или 100% | Откалибровать датчики |
| Быстро садится | Увеличить интервал сна |
