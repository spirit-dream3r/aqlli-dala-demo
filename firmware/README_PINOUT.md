# ESP32-C3 Super Mini + Датчики влажности почвы v1.2

## 📋 Распиновка

### ESP32-C3 Super Mini Pinout

```
                    ESP32-C3 Super Mini
                    ┌─────────────────┐
                    │                 │
         3.3V ──────┤ 5V  GND  3.3V  ├─── GPIO4 (D4) ──── Датчик 1 (A0)
                    │                 │
         VIN  ──────┤ D4   D5   D6   ├─── GPIO5 (D5) ──── Датчик 2 (A0)
                    │                 │
         GND ───────┤ GND  RST  D10  ├─── GPIO6 (D6) ──── (свободный)
                    │                 │
                    └─────────────────┘
```

### Подключение датчиков влажности почвы v1.2

| Датчик v1.2 | ESP32-C3 Super Mini | GPIO | Описание |
|-------------|---------------------|------|----------|
| VCC (красный) | 3.3V | - | Питание 3.3V |
| GND (черный) | GND | - | Земля |
| A0 (желтый) | D4 | GPIO4 | Аналоговый выход датчик 1 |
| A0 (желтый) | D5 | GPIO5 | Аналоговый выход датчик 2 |
| DO (синий) | - | - | Не используется |

> **Примечание:** Датчики v1.2 работают от 3.3V-5V. Рекомендуется 3.3V для совместимости с логическими уровнями ESP32-C3.

### Подключение аккумулятора 18650

```
    Аккумулятор 18650          Плата защиты TP4056         ESP32-C3 Super Mini
    ┌─────────────┐            ┌─────────────────┐         ┌─────────────────┐
    │     + ──────┼────────────┤ B+          P+  ├─────────┤ VIN (или 5V)    │
    │             │            │                 │         │                 │
    │     - ──────┼────────────┤ B-          P-  ├─────────┤ GND             │
    └─────────────┘            └─────────────────┘         └─────────────────┘
```

#### Вариант 1: Через плату защиты TP4056 (РЕКОМЕНДУЕТСЯ)

| Компонент | Подключение |
|-----------|-------------|
| Аккумулятор + | B+ на TP4056 |
| Аккумулятор - | B- на TP4056 |
| P+ на TP4056 | VIN на ESP32-C3 |
| P- на TP4056 | GND на ESP32-C3 |
| Micro-USB на TP4056 | Для зарядки от USB |

**Преимущества:**
- ✅ Защита от переразряда
- ✅ Защита от перезаряда
- ✅ Защита от КЗ
- ✅ Встроенная зарядка от USB

#### Вариант 2: Прямое подключение (только если есть BMS в аккумуляторе)

| Аккумулятор | ESP32-C3 |
|-------------|----------|
| + (плюс) | VIN (или 5V пин) |
| - (минус) | GND |

> ⚠️ **ПРЕДУПРЕЖДЕНИЕ:** Используйте только с защищенными аккумуляторами (с встроенной BMS платой)!

## 🔧 Настройка прошивки

### 1. Установка Arduino IDE и библиотек

1. Установите [Arduino IDE](https://www.arduino.cc/en/software)
2. Добавьте плату ESP32-C3:
   - Откройте `File` → `Preferences`
   - В "Additional Boards Manager URLs" добавьте:
     ```
     https://espressif.github.io/arduino-esp32/package_esp32_index.json
     ```
3. Откройте `Tools` → `Board` → `Boards Manager`
4. Найдите "esp32" и установите "esp32 by Espressif Systems"
5. Выберите плату: `Tools` → `Board` → `ESP32C3 Dev Module`

### 2. Настройка параметров WiFi

Откройте файл `esp32c3_soil_sensor.ino` и измените:

```cpp
const char* WIFI_SSID = "YOUR_WIFI_SSID";        // Ваше WiFi имя
const char* WIFI_PASSWORD = "YOUR_WIFI_PASSWORD"; // Ваш WiFi пароль
const char* SERVER_URL = "http://your-server-ip:8000/sensor-data";
const String DEVICE_ID = "ESP32C3_SOIL_01";
```

### 3. Калибровка датчиков

Для точных измерений откалибруйте датчики:

```cpp
const int CALIBRATION_DRY = 3200;   // Значение в сухом воздухе
const int CALIBRATION_WET = 1500;   // Значение в воде
```

**Процесс калибровки:**
1. Загрузите тестовый скетч для чтения ADC
2. Поместите датчик в сухой воздух → запишите значение
3. Поместите датчик в стакан с водой → запишите значение
4. Вставьте значения в прошивку

### 4. Настройка интервала измерений

```cpp
const int SLEEP_DURATION_MINUTES = 10;  // Интервал между измерениями
```

Рекомендуемые значения:
- 🌱 Активный мониторинг: 5-10 минут
- 🔋 Экономия батареи: 30-60 минут
- ⚡ Максимальная экономия: 120+ минут

## 📊 Потребление энергии

| Режим | Ток | Примечание |
|-------|-----|------------|
| Активный (WiFi) | ~100-150 mA | Во время измерения и отправки |
| Глубокий сон | ~10-50 μA | Между измерениями |
| **Среднее (10 мин)** | ~0.5 mA | Зависит от интервала |

**Расчет времени работы от 18650:**
- Емкость 18650: 2000-3500 mAh
- При среднем потреблении 0.5 mA: **4000-7000 часов** (~6-12 месяцев)

## 💾 Загрузка прошивки

1. Подключите ESP32-C3 Super Mini к компьютеру через USB-C кабель
2. В Arduino IDE выберите:
   - `Tools` → `Board` → `ESP32C3 Dev Module`
   - `Tools` → `Port` → (ваш COM порт)
3. Нажмите кнопку **Upload** (→)
4. Если не загружается, зажмите кнопку **BOOT** на плате и нажмите **RESET**

## 🔍 Мониторинг

Откройте **Serial Monitor** (115200 бод) для просмотра:
- Показаний датчиков
- Статуса подключения WiFi
- Напряжения батареи
- Статуса отправки данных

## 🛠️ Устранение неполадок

| Проблема | Решение |
|----------|---------|
| Не подключается к WiFi | Проверьте SSID и пароль, убедитесь что WiFi 2.4GHz |
| Показания датчиков 0% или 100% | Откалибруйте датчики, проверьте подключение |
| Быстрый разряд батареи | Увеличьте SLEEP_DURATION_MINUTES |
| Не загружается прошивка | Зажмите BOOT при подключении USB |
